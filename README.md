# 🚀 Modern Glassmorphism Portfolio

A high-performance, minimalist developer portfolio built with **Laravel 13** and **Tailwind CSS v4**. This project showcases advanced backend architecture, real-time status updates, and a modern "Frosted Glass" UI/UX.

## 🛠 Tech Stack
- **Framework:** Laravel 13 (Blade & Alpine.js)
- **Styling:** Tailwind CSS v4 (Glassmorphism & Bento Grid)
- **Environment:** Docker (Laravel Sail)
- **Database:** MySQL
- **Icons:** Lucide Icons
- **Animations:** Alpine.js + Tailwind Transitions

## Key Features
- **Modern UI:** Glassmorphism design with animated mesh gradients.
- **Bento Grid:** A geometric, minimalist project showcase layout.
- **Responsive:** Optimized for all screen sizes.
- **Optimized for Shared Hosting:** Designed to run efficiently on standard servers.

## 🚀 Local Development

To get started with this project locally using **Laravel Sail**:

1. **Clone the repo:**
   ```bash
   git clone [https://github.com/itsmrsarfaraz/portfolio.git](https://github.com/itsmrsarfaraz/portfolio.git)
   cd portfolio
   ```

2. **Install dependencies:**
    ```bash
    composer install
    npm install
    ```

3. **Start the environment:**
    ```bash
    ./vendor/bin/sail up -d
    ```
    
4. **Run migrations:**
    ```bash
    ./vendor/bin/sail artisan migrate
    ```


5. **Compile assets:**
   ```bash
   ./vendor/bin/sail npm run dev
   ```
